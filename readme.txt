Adds a dashboard support system for Nations PC Solutions members WP Powered Website Admin Dashboard and widget area (under Active Development).

== Description ==

Adds support features and tools for members who use a Nations PC Solutions Powered WP WebSite, includes support tickets, lessons, help, guides, and more tools for NPS Members Only...

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All Done :) you can access the the widget and Dash Page from the WP admin Dashboard .
